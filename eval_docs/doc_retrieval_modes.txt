Режимы retrieval: bm25 (только лексический поиск), vector (только семантический), hybrid (fusion BM25 + vector). Rerank улучшает порядок после первичного поиска. top_k — число чанков. use_adaptive_k — автоматическая подстройка. Фильтры: project_id (коллекция), tenant_id. Контекст передаётся в LLM с нумерацией [1], [2] для цитирования.
